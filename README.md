# gym_sandbox

## Quickstart

### Host installs
sudo apt install python3-venv

### Create venv
python3 -m venv .venv

### Activate
source .venv/bin/activate

### Venv installs
pip install "gymnasium[atari]"
pip install autorom[accept-rom-license]
AutoROM --accept-license
pip install moviepy

pip3 install matplotlib
pip install seaborn

### Deactivate
deactivate
